Перевірки:
1) .tagName - визначає тег елемнту (div, span, label і т.д.)
2) Array.isArray() - перевірити чи є змінна масивом
3) .className - отримати назву класу

Вибірка
1) .closest('.acardion_items') - знайти перший батьківський елемент

Координати
1) .offsetTop -  поизиція елементу (offsetLeft)
2) .getBoundingClientRect() визначити координати

Додавання 
1) .setAttribute(name, value) - дадати атрибут
2) .getAttribute('name') - отримати атребут
3) classList - маніпуляції з class
        add - додати
        contains - перевірити чи місить класс
        remove - видаляє класс
        повний перелік з маніпуляціями класів тут https://www.w3schools.com/jsref/prop_element_classlist.asp



Перетворення
